BUILD:- 

Create a catkin workspace and copy this project and then do catkin_make.

Launch:- 

roslaunch ugv model.launch to launch gazeboo and Rviz with TF

roslauncg ugv turtlebot3_teleop_key.launch to launch for teleoperation

Scripts:-

python draw_a_square.py to draw a square

python move_circle_server.py to draw a server for circle 
python move_circle_client.py to run the client for circle.
